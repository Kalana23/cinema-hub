# 🎬 CINEMA HUB

A modern, responsive entertainment database built strictly with HTML5 and CSS3. Features a sleek Glassmorphism UI, embedded video trailers, and smooth horizontal scroll interactions without a single line of JavaScript.

## ✨ Features

* **Premium Glassmorphism UI:** Modern frosted-glass navigation bar and floating developer badges.
* **Dynamic Layouts:** Horizontal scrolling for movie rows and responsive grid for the Watchlist.
* **Embedded Trailers:** Watch official YouTube trailers directly on the info pages.
* **Zero JavaScript UI:** Entirely built using pure HTML and CSS, demonstrating advanced styling and layout techniques.
* **Fully Responsive:** Adapts beautifully to different screen sizes.

## 📂 Project Structure

* `index.html` - Home page featuring trending movies.
* `tv-shows.html` - TV shows discovery page.
* `people.html` - Top creators and actors page.
* `watchlist.html` - Personal saved items.
* `*-info.html` - Detailed pages with embedded trailers.
* `style.css` - Global stylesheet containing all UI designs.
